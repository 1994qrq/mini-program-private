### 4. 图文特殊库权限 ✅

**实现位置**：
- `src/utils/familiar-local.ts` - 权限管理逻辑
- `src/pages/sub-page/stepTask/shuxi/stage0.ts` - 激活触发

**功能描述**：
- 问3选择"是"时自动激活图文特殊库权限
- 权限信息存储在全局storage中
- 6-9天后随机刷新特殊内容
- 新任务的权限会覆盖旧任务的权限
- 提供权限查询和刷新接口

**关键函数**：
- `activateImageTextSpecialPermission()` - 激活权限
- `getImageTextSpecialPermission()` - 获取权限信息
- `refreshImageTextSpecialContent()` - 刷新特殊内容
- `deactivateImageTextSpecialPermission()` - 取消权限

**数据结构**：
```typescript
interface ImageTextSpecialPermission {
  enabled: boolean;    // 是否激活
  content: string;     // 特殊内容
  activatedAt: number; // 激活时间
  refreshAt: number;   // 下次刷新时间
  taskId: string;      // 激活该权限的任务ID
}
```
