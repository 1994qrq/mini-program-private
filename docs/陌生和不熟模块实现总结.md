## 使用说明

### 创建陌生模块任务

```typescript
import { initSmLocal, createTask } from '@/utils/stranger-local';

// 初始化
initSmLocal();

// 创建任务
const result = createTask({
  name: '张先生的咨询',
  durationDays: 5
});

if (result.ok) {
  console.log('任务创建成功:', result.task);
  // 跳转到对话页
  uni.navigateTo({
    url: `/pages/sub-page/stepTask/round-stranger?taskId=${result.task.id}`
  });
}
```
